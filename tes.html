Kali ini, saya akan menjelaskan mengenai bahasa pemrograman Perl.

Perl adalah singkatan dari Practical Extraction and Report Language. Bahasa pemrograman ini diciptakan oleh Larry Wall pada tahun 1986 yang awalnya digunakan sebagai perangkat lunak yang digunakan untuk menulis program di lingkungan UNIX. Ekstensi file program Perl adalah (.pl) yang dapat dijalankan menggunakan software Perl. Juga dapat dijalankan menggunakan terminal dengan command :

perl program.pl
